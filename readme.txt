This plugin inserts an IndiTip tip-button into your WordPress posts.

== Description ==

IndiTip is an India-only service which lets people tip for good content on the Internet. We provide tip-buttons which allow your readers to tip you five rupees in appreciation of your work. 

You need an Indian bank account or mobile wallet to receive tips. If you're receiving tips as an individual, that's usually all you need. If you're a registered company we will ask for your PAN card and registration deed.

== Installation ==

Installing "Tip button from IndiTip" can be done either by searching for "Tip button from IndiTip" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is my userid? =

Your userid is assigned by IndiTip when you create an account on inditip.com. You can find it in your settings (https://inditip.com/#settings).

= Can I place the plugin somewhere else on my blog? =

Yes, by using the [inditip] shortcode. This needs to be enabled in the plugin's settings.

= Can my international readers tip me? =

IndiTip only processes domestic (INR) transactions at this time. All payments are processed by PayTM.
